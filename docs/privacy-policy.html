Privacy Policy

Last updated: April 19, 2026

This application ("Training Data Pipeline") is a personal project created and operated by Martin J. It connects to WHOOP and TrainingPeaks APIs to retrieve personal health and training data for private analysis.

Data Collection

This application accesses health and fitness data (including recovery scores, sleep data, heart rate, and workout data) solely from the account of the application owner. No data is collected from any other users.

Data Use

Data retrieved through this application is used exclusively for personal training analysis. It is not sold, shared, or distributed to any third parties.

Data Storage

All data is stored locally on the owner's personal device. No data is stored on external servers or cloud services.

Third-Party Services

This application connects to the WHOOP API and TrainingPeaks API. Use of those services is governed by their respective privacy policies.
